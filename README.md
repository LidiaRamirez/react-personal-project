### Proyecto Personal

Este proyecto es realizado basado en:
- React
- Sass
- Atomic Design
- BEM

## Instalaciones de paquetes 

### Para inicio de proyecto con react
~~~
npx create-react-app my-app
~~~

### Para uso de Sass
~~~
npm i node-sass
~~~
